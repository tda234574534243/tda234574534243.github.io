# Portfolio — Trần Đức Anh

Trang portfolio cá nhân dùng để giới thiệu chuyên môn, dự án và chứng chỉ. Giao diện được tối giản, phù hợp để deploy bằng GitHub Pages.

## Link website

- https://tda234574534243.github.io/

## Mục lục

- Giới thiệu — Tóm tắt bản thân và thế mạnh.
- Dự án — Danh sách dự án tiêu biểu kèm mô tả và link nguồn khi có.
- Chứng chỉ — Các chứng chỉ, thành tích liên quan.
- Liên hệ — Email và GitHub.

## Những thay đổi chính

- Tách CSS ra `assets/css/style.css` để dễ tùy chỉnh.
- Cải thiện bố cục: thêm thanh điều hướng, lưới dự án, và khung hiển thị thông tin.

## Xem trang trên máy local

1. Mở trực tiếp `index.html` trong trình duyệt, hoặc chạy server tĩnh:

```bash
python -m http.server 8000
# mở http://localhost:8000
```
